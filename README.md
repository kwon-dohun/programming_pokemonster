# 🐾 포켓몬 미니 배틀 게임 (Pokemon Mini Battle Game)

![Python](https://img.shields.io/badge/Python-3.13-blue)
![Status](https://img.shields.io/badge/Status-OnProgress-brightgreen)

콘솔 환경에서 즐기는 **턴제 포켓몬 미니 배틀 게임**이다.  
스타터 포켓몬을 선택하고 야생 포켓몬과 배틀하며 성장한다.

단순한 공격 반복이 아니라 **확률 기반 공격**, **타입 상성**,  
**목숨(Life) · 알약(Pill) 관리 시스템**을 포함한 미니 게임이다.

---

## 프로젝트 개요

- **플랫폼**: Python 콘솔 게임
- **장르**: 턴제 배틀 RPG
- **목표**: 조건문 중심의 단순 게임이 아닌    **상태 관리 · 확률 · 데이터 분리 설계**를 적용한 미니 게임 구현

---

## 주요 기능

### 1. 기본 게임 플레이
- **스타터 포켓몬 선택**
  - 피카츄 / 파이리 / 꼬부기 / 이상해씨
- **야생 포켓몬과 1:1 턴제 배틀**
- **경험치 획득 → 레벨업 → 진화**



### 2. 전투 시스템

#### **공격 방식**
- **안정 공격**
  - 명중률 **100%**
  - 기본 공격력 기반 데미지

- **강력 공격 (확률형)**
  - **10%** : 매우 강한 공격
  - **50%** : 강한 공격
  - **40%** : 공격 실패

공격 실패 시 : 공격을 실패했다!


#### **타입 상성**
- 타입에 따라 데미지 배율 적용
- 예시:
  - 전기 → 물 : **효과가 굉장했다**
  - 전기 → 땅 : **공격이 통하지 않음**

#### **전투 중 회복**
- 전투당 **최대 3회**
- 체력 일부 회복

#### **도망**
- 성공 확률 존재
- 성공 시 전투 즉시 종료



### 3. 목숨 (Life) 시스템
- **최대 목숨: 5**
- 배틀 결과에 따른 소모
  - **패배**: `-1`
  - **도망 성공**: `-0.5`
- 목숨이 **0이면 배틀 불가**
- **1시간마다 목숨 1 자동 회복**
- 목숨이 모두 소진되면 **회복 대기 메시지** 출력



### 4. 알약 (회복 아이템)
- **하루 최대 10개**
- 상태 메뉴에서 사용 가능
- 사용 시 **HP 100% 완전 회복**
- 날짜가 바뀌면 **자동 초기화**



### 5. 도감 시스템
- **타입별 상성 정보 제공**
- 각 타입에 대해:
  - **강한 타입**
  - **약한 타입**
  - **설명 문구**
- 데미지 계산 로직과 분리된 **설명 전용 도감**

## 프로젝트 구조

```text
programming_pokemonster
├─ main.py                    # 게임 실행 진입점
└─ core/
    ├─ battle.py              # 전투 로직
    ├─ pokemon.py             # 포켓몬 클래스 (공격, 레벨업, 진화)
    ├─ player.py              # 플레이어 상태 (목숨, 알약)
    ├─ data.py                # 게임 데이터 및 도감
    └─ type_effectiveness.py  # 타입 상성 계산

```
